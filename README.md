# Hello World Web Application

Веб-приложение на Express и TypeScript с HTML страницей "Hello, World!".

## Инсталляция

```bash
npm install
```

## Сборка

```bash
npm run build
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Разработка (с автоматической перезагрузкой)

```bash
npm run dev
```

## Структура проекта

- `src/index.ts` — Express сервер
- `public/index.html` — HTML страница с приветствием
- `public/` — статические файлы (CSS, JS, изображения)
